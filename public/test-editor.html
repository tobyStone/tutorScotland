Instructions

How to use the Visual Editor:

  1. Login as Admin: Go to login page and login with admin credentials
  2. Enable Edit Mode: Click the "Edit Mode" button in the top-right corner or press Ctrl+E
  3. Edit Content: Hover over any element to see edit controls, then click "Edit"
  4. Save Changes: Use the modal to modify content and save changes
  5. Preview: Use the preview button to see changes temporarily
  6. Restore: Use the restore button to revert to original content

Supported Content Types:

  • Text: Simple text content (headings, paragraphs)
  • HTML: Rich HTML content with formatting
  • Images: Image URLs and alt text
  • Links: Link URLs and display text
